# AutoYoloTrainer 의존성 설치 가이드

## 1. 자동 설치
다른 PC로 프로그램 폴더를 옮긴 뒤 아래 파일을 실행합니다.

```text
install_dependencies.bat
```

배치 파일 내부 문구는 Windows `cmd.exe` 호환성을 위해 영어로 작성되어 있습니다.

## 2. 설치되는 기본 패키지
`Code/requirements.txt` 기준 패키지:

- `PyQt6`
- `PyInstaller`
- `psutil`
- `nvidia-ml-py`
- `ultralytics`

`ultralytics` 설치 과정에서 YOLO 학습에 필요한 `torch`, `opencv-python` 등 하위 의존성이 함께 설치됩니다.

## 3. 수동 설치
자동 설치가 실패하면 `AutoYoloTrainer/Code` 폴더에서 실행합니다.

```powershell
py -3 -m pip install --upgrade pip
py -3 -m pip install -r requirements.txt
```

## 4. CUDA 확인
설치 후 다음 값이 `True`이면 PyTorch가 CUDA GPU를 사용할 수 있습니다.

```powershell
py -3 -c "import torch; print(torch.cuda.is_available())"
```

`False`이면 CPU 학습만 가능하거나 CUDA용 PyTorch 재설치가 필요합니다.

## 5. 설치 확인
아래 명령이 오류 없이 실행되어야 합니다.

```powershell
py -3 -c "import PyQt6, psutil, pynvml, ultralytics, cv2, torch; print('OK')"
```

## 6. 자주 발생하는 문제

### Python을 찾지 못하는 경우
- Python 3.10 이상 설치
- 설치 시 `Add Python to PATH` 체크
- 설치 후 새 터미널에서 다시 실행

### GPU가 표시되지 않는 경우
- NVIDIA Driver 설치 확인
- `nvidia-ml-py` 설치 확인
- 노트북이면 외장 GPU 활성화 상태 확인

### GPU 학습이 되지 않는 경우
- `torch.cuda.is_available()` 확인
- CUDA 지원 PyTorch 설치 필요
- NVIDIA Driver와 PyTorch CUDA 버전 호환성 확인

### EXE는 실행되지만 학습이 실패하는 경우
- EXE 내부에 Ultralytics를 포함하지 않음
- 학습, export, detect는 시스템 Python 패키지를 사용
- 따라서 `install_dependencies.bat` 실행 필요
