# AutoYoloTrainer 실행 환경 요구사항

## 1. 운영체제
- Windows 10/11 권장
- NVIDIA GPU 사용 시 NVIDIA Driver 설치 필요

## 2. Python
- 권장: Python 3.10
- 허용: Python 3.10 이상
- conda 사용 시 권장 환경명: `yolo_stable`

## 3. 필수 Python 패키지
`Code/requirements.txt` 기준:

- `PyQt6`
- `PyInstaller`
- `psutil`
- `nvidia-ml-py`
- `ultralytics`

YOLO 학습에는 추가로 아래 패키지가 필요합니다.

- `torch`
- `torchvision`
- `torchaudio`
- `opencv-python`

## 4. 실행 방식
일반 실행:

```text
AutoYoloTrainer.exe
```

conda 환경 지정 실행:

```text
start_autoyolotrainer_conda.bat
```

특정 conda env 경로 지정:

```text
start_autoyolotrainer_conda.bat "C:\Users\innom\miniconda3\envs\yolo_stable"
```

## 5. 단독 실행 정책
- `training_runner.py`, `model_tool_runner.py`는 exe 내부에 포함됩니다.
- `Code` 폴더 없이도 exe 실행, 학습, export, detect가 가능해야 합니다.
- 단, 외부 Python 환경에는 YOLO 관련 패키지가 설치되어 있어야 합니다.

## 6. conda 실행 정책
- bat 파일은 `conda activate`를 하지 않습니다.
- exe 자체는 깨끗한 환경에서 실행합니다.
- 학습/export/detect 자식 프로세스만 지정 conda Python을 사용합니다.
- 이 방식은 PyInstaller Python DLL과 conda Python DLL 충돌을 줄이기 위한 정책입니다.

## 7. 보안 정책 주의
- Device Guard / WDAC / AppLocker가 적용된 PC에서는 exe 실행이 차단될 수 있습니다.
- 이 경우 코드 수정이나 bat로 우회할 수 없으며, 보안 담당자의 허용 정책이 필요합니다.
